Plan de Mejoramiento

Proyecto con Vite, Base de Datos y SQL

Ruta de recuperacion academica para fortalecer competencias tecnicas en desarrollo web y gestion de datos.

Docente: Ing. Edwin Marin
Institucion: SENA
Duracion: 2 semanas
Objetivo

Objetivo General

Construir una aplicacion web funcional con Vite conectada a base de datos, aplicando consultas SQL para operaciones de registro, consulta, actualizacion y eliminacion de informacion.

Competencias a desarrollar

  • Crear proyectos modernos con Vite.
  • Modelar y crear tablas en SQL.
  • Conectar frontend con base de datos.
  • Implementar CRUD completo.

Resultado esperado

  • Aplicacion funcional y estable.
  • Consultas SQL documentadas.
  • Repositorio con evidencias tecnicas.
  • Presentacion final del proyecto.
Alcance

Entregables Obligatorios

  • Repositorio GitHub con codigo fuente.
  • Carpeta sql/ con schema.sql, seed.sql y consultas.sql.
  • README con instalacion, ejecucion y descripcion tecnica.
  • Video de 3 a 5 minutos demostrando funcionamiento.

Requisitos minimos

  • Minimo 3 tablas relacionadas.
  • Minimo 8 consultas SQL diferentes.
  • CRUD funcional desde interfaz web.
  • Aplicacion ejecutando sin errores criticos.
Cronograma

Semana 1

Dia 1: Configuracion de entorno y creacion del proyecto Vite.
Dia 2: Definicion del problema y modelado de base de datos.
Dia 3: Creacion de tablas e insercion de datos de prueba.
Dia 4: Consultas SQL: SELECT, WHERE, ORDER BY, JOIN.
Dia 5: Conexion de la aplicacion y visualizacion de datos.
Cronograma

Semana 2

Dia 6: Formulario de creacion de registros (INSERT).
Dia 7: Edicion de registros (UPDATE).
Dia 8: Eliminacion controlada de registros (DELETE).
Dia 9: Busqueda, filtros y mejora de experiencia.
Dia 10-12: Validaciones, manejo de errores y ajustes UI.
Dia 13: Documentacion y evidencias.
Dia 14: Sustentacion y entrega final.
Evaluacion

Rubrica de Entrega (100 puntos)

Criterio Peso Meta esperada
Configuracion del proyecto 10% Estructura clara y app ejecutando correctamente.
Diseno de base de datos 15% Tablas y relaciones coherentes.

Integracion app + base de datos 15% Conexion funcional y estable.
