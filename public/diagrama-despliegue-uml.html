UsuarioNavegador --> HostingFrontend : HTTPS
HostingFrontend --> AppEBS : entrega artefactos
AppEBS --> SupabaseAPI : Supabase JS Client (anon key)
SupabaseAPI --> SupabaseDB : consultas SQL / RLS
SupabaseAPI --> Backups : tarea programada
GitHubRepo --> CICD : trigger por push
CICD --> HostingFrontend : despliegue automatico

Si estas en desarrollo con Vite: ejecuta npm run dev y entra a http://localhost:5173/diagrama-despliegue-uml.html.
